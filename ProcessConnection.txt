Межпроцессное взаимодействие:

1. Используемые обозначения:
  1.1. Самосвал - #2;
  1.2. Регулировщик - #3;
  
2. Введение в методы взаимодействия в данной задаче:
  2.1. Очереди сообщений:
    2.1.1. Самосвал - Регулировщик:
      2.1.1.1. Регулировщик выступает выступает в качестве сервера, Самосвал - в качестве клиента;
      2.1.1.2. Самосвал посылает сообщение Регулировщику, расположенному с ним по одну сторону моста, после подъезда к мосту, а Регулировщик принимает его.
  2.2. Семафоры:
    2.2.1. Самосвал - Регулировщик:
      2.2.1.1. Регулировщик разрешает Самосвалу проезд по мосту, устанавливая заданное значение семафора;
      2.2.1.2. После проезда по мосту Самосвал меняет значение семафора на исходное (исходное значение == 1).
    2.2.2. Регулировщик - Регулировщик:
     2.2.2.1. Применяется для определения того, какой Регулировщик руководит движением на мосту в данный момент.

3. Необходимые данные для корректного взаимодействия, содержащиеся в процессах:
  3.1. Самосвал:
    3.1.1. Собственный ключ семафора самосвала;
    3.1.2. Ключ очереди сообщений первого и второго регулировщиков;
    3.1.3. Масса самосвала;
    3.1.5. Расположение самосвала относительно моста.
  3.2. Регулировщик:
    3.2.1. Собственный ключ очереди сообщений;
    3.2.2. (Собственный) Ключ семафора регулировщиков;
    3.2.3. Расположение регулировщика относительно моста ({0, 2}).

4. Порядок взаимодействия с "соседними" процессами:
  4.1. Самосвал - Регулировщик:
    4.1.1. После подъезда к мосту Самосвал посылает соответствующему Регулировщику сообщение по форме "%Ключ семафора Самосвала% %Масса Самосвала%", а затем ждёт разрешение на проезд от Регулировщика;
    4.1.2. Если Регулировщик разрешает Самосвалу движение по мосту, то меняет значение его семафора на 0 (уменьшает на 1).
    4.1.3. После выезда с моста Самосвал меняет значение семафора на 2 (добавляет 2);
    4.1.4. Регулировщик меняет значение семафора Самосвала на 1 (уменьшает на 1);
  4.2. Регулировщик - Регулировщик:
    4.2.1. Для передачи управления движением Регулировщик меняет значение семафора на номер другого регулировщика, а затем ждёт, пока семафор не примет значение, соответствующее его номеру.
